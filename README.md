# My-own-Data-Analysis-repository
Bienvenido a mi repositorio de proyectos realizados en análisis de datos.
Soy Luis Andrés Aponte. Ingeniero mecánico con experiencia en docencia,  consultoría de ERPs y conocimientos en análisis de datos. Apasionado por la tecnología y por el continuo crecimiento, avalado por medio de decisiones a conciencia, mediante el uso adecuado de los datos y la información.
Para conocer más detalle de cada proyecto depositado en este repositorio, por favor entra a la carpeta del proyecto que quieras entrar y ahí encontrarás los archivos requeridos para probar cada proyecto y un README particular con indicaciones particulares para cada proyecto.
